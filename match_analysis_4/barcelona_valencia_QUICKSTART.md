## Troubleshooting

**Issue**: "Module not found"  
**Fix**: Run `pip install pandas numpy matplotlib seaborn jupyter`

**Issue**: "File not found"  
**Fix**: Ensure `16157.json` is in the same directory

**Issue**: Jupyter won't start  
**Fix**: Try `pip install --upgrade jupyter notebook`
